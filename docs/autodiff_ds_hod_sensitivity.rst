HOD Parameter Sensitivity of :math:`\Delta\Sigma` — All Models
===============================================================

This page mirrors the :math:`w_p` all-models analysis (see
:doc:`autodiff_hod_sensitivity`) for the **excess surface mass density**
:math:`\Delta\Sigma(R)`.  For each of the six HOD models fitted to the
BGS M10 sample, sensitivities are computed via numerical **centred finite
differences** (1% relative step) through the full **1-halo + 2-halo** halo
model (:class:`~hod_mod.galaxies.clustering.FullHaloModelPrediction`,
More+2015, NFW profile, Diemer+19 concentration–mass relation).

**Parameters**: best-fit MAP values from the BGS M10 LS10 DR10 multi-probe fit
(:math:`r_p > 0.3\,h^{-1}{\rm Mpc}`, :math:`z_{\rm eff} = 0.136`), stored in
``results/bgs_multiprobe/mstar10.0_wp_{model}_nfw_rp300/map_result.json``.

All figures and tables are generated by
``hod_mod/scripts/sensitivity/demo_sensitivity_ds_all_hod.py``::

    python hod_mod/scripts/sensitivity/demo_sensitivity_ds_all_hod.py \
        --output results/showcase/fig_sensitivity_ds

Overview: What Changes Across HOD Models
------------------------------------------

The logarithmic response of :math:`\Delta\Sigma(R)` to a cosmological parameter
is **model-dependent** in amplitude but relatively stable in shape: the
effective galaxy–matter cross-power depends on :math:`b_{\rm eff}` (one factor,
vs :math:`b_{\rm eff}^2` for :math:`w_p`), so the cosmological responses are
slightly larger than in :math:`w_p` (see :doc:`autodiff_ds_sensitivity`).

With the full 1h+2h model, :math:`\mathcal{S}_{\sigma_8}` is now strongly
**scale-dependent** (peaking near :math:`R \sim 3\,h^{-1}{\rm Mpc}`) and
**near model-independent** (varying only ±2% across models at :math:`R=1`).
This replaces the 2-halo-only value of :math:`\approx 1.79` (constant) with
a physically correct 1-halo–dominated response of :math:`\approx 7.0\text{–}7.1`
at :math:`R = 1\,h^{-1}{\rm Mpc}`.  :math:`\mathcal{S}_{\Omega_m}` remains
positive down to :math:`R \sim 20\,h^{-1}{\rm Mpc}`, still contrasting with
:math:`w_p`.

Cosmological Parameters: Model Comparison
-------------------------------------------

.. figure:: ../results/showcase/fig_sensitivity_ds_cosmo_comparison.png
   :width: 95%
   :align: center

   Logarithmic response :math:`\mathcal{S}_\theta(R)` for
   :math:`\Omega_m`, :math:`\sigma_8`, :math:`h`, and :math:`n_s` across all
   nine HOD models.  The close overlap between models confirms that the
   **cosmological shape information in :math:`\Delta\Sigma` is largely
   model-independent** at Planck 2018 default parameters.

Key findings:

- :math:`\mathcal{S}_{\sigma_8}` is now **scale-dependent** (1h+2h), peaking at
  :math:`\approx 9.6\text{–}9.8` near :math:`R \sim 3\,h^{-1}{\rm Mpc}` (1-halo
  dominated) and settling to :math:`\approx 6.0` at :math:`R = 30\,h^{-1}{\rm Mpc}`.
  It is near-identical across all six models (within ±2% at each scale).
- :math:`\mathcal{S}_{\Omega_m}` curves overlap within ±0.06 across all models
  at all scales, confirming model-independence.
- The small spread in :math:`\mathcal{S}_{\sigma_8}` at fixed scale comes from
  slightly different HOD threshold masses setting different 1-halo amplitudes.
- Both :math:`\Delta\Sigma` and :math:`w_p` (see :doc:`autodiff_hod_sensitivity`)
  show the same model-near-independence of cosmological sensitivities with 1h+2h.

Degeneracy Patterns
--------------------

The Pearson correlation matrices in the per-model figures reveal degeneracies
in the response space.  Common patterns across all models:

- :math:`h` and :math:`\Omega_m` are **strongly correlated**
  (:math:`r \approx 0.98`) — both enter through the matter transfer function
  shape parameter :math:`\Omega_m h^2`.
- :math:`n_s` and :math:`h` are **moderately correlated**
  (:math:`r \approx 0.96`) — both tilt the power spectrum.
- :math:`\sigma_8` is **correlated with** :math:`\Omega_m` and :math:`h`
  (:math:`r \approx 0.99`) with the 1h+2h model — unlike the 2-halo-only
  regime where :math:`\sigma_8` was orthogonal (pure amplitude).  The 1-halo
  term couples amplitude and shape through the HMF and concentration.
- Within each HOD, the :math:`\log M_1` and :math:`\alpha` (satellite slope)
  parameters are typically degenerate (:math:`r > 0.90`) because both set
  the satellite fraction and hence :math:`b_{\rm eff}`.

These degeneracies motivate combining :math:`\Delta\Sigma` with :math:`w_p`
for parameter estimation (SNR forecast page coming soon): the
ratio :math:`\Delta\Sigma / w_p \propto 1/b_{\rm eff}` breaks the
galaxy bias degeneracy, enabling simultaneous constraints on
:math:`\sigma_8`, :math:`\Omega_m`, and :math:`b_{\rm eff}`.
