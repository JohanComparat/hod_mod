# Pinned Python dependencies for the GNU Guix install of hod_mod.
#
# These versions reproduce the validated science environment (the conda
# `hod_mod` env). Pinning matters: an unpinned `pip install camb` pulls a newer
# camb whose `set_matter_power` relies on the implicit 1-element-array -> scalar
# conversion that NumPy >= 2.4.0 removed, producing
#   TypeError: only 0-dimensional arrays can be converted to Python scalars
# at camb/model.py:691 and failing ~140 tests. camb 1.4.0 avoids that path.
#
# camb 1.4.0 is source-only and supports Python <= 3.11, so install inside the
# Guix time-machine shell (channels.scm -> Python 3.11), where the Guix
# gfortran-toolchain compiles it. See INSTALL_GUIX.md.
